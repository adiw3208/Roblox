# Smart Auto Clicker For Roblox | Stunt

## Requirment :
1. Python 3.11.x
2. `py -3.11 -m venv venv311`
3. `venv311\Scripts\activate`
4. `pip install customtkinter pywin32 keyboard psutil pyinstaller`
5. `pyinstaller --onefile --windowed --icon=icon.ico --version-file version.txt app.py`
